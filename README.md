# 💾 SaveMyDB

A Spreadsheet-to-Database Synchronization Platform.

## 🚀 Features
- Google Sheets to MySQL sync
- JWT Authentication
- Auto sync every 30 minutes
- React Native mobile app
- Web dashboard
- Google Sheets extension

## 🛠️ Setup
1. Clone: git clone https://github.com/rsmmlkhader-star/savemydb.git
2. Install: pip install -r requirements.txt
3. Copy: cp .env.example .env
4. Run: python -m flask run

## 🌐 Live API

- POST /api/auth/login
- GET  /api/health

- POST /api/sync/import
